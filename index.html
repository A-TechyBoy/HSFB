HSFB - High Seas for Beginners

Information for new HighSeas hackers

1. Stuck in Shop/Shipyard?

First try clearing your cookies. If that doesn't work:

Shop Issue

  • Re-verify through the Verification Link

Shipyard Issue

  • Set up hackatime

2. Curses and Blessings Explained

Curses (-50% doubloons)

  • Given for minimal effort votes or using dots to reach word count

Blessings (+20% doubloons)

  • Earned through detailed votes with suggestions and insights

3. Track Your Shipment

  • Check the Shipment Viewer for items shipped from HackClub's warehouse
  • Note: Items from other vendors won't appear here

4. Hackatime Data Collection

  • Based on Wakatime's privacy policy
  • Collects usage data but not source code
  • See policy for complete details

5. Non-VS Code Setup

  • Visit wakatime.com/plugins
  • Install your editor's plugin
  • Get Hackatime config from: HighSeas → SignPost → Hackatime instructions
  • Replace Wakatime config with Hackatime config in your editor

6. Doubloon Calculations

  • Based on project hours and ranking against similar projects
  • Average rates per hour:
    • Normal: 5-25 doubloons
    • With curse: 2.5-12.5 doubloons
    • With blessing: 6-30 doubloons
